# Image Preprocessing Assignment

This repository contains:
- Five image preprocessing techniques
- Implementation in PyTorch
- Code ready for training and visualization

## Files
- `preprocessing.py`: Main preprocessing script
- `requirements.txt`: Libraries needed
- `images/`: Folder where generated images will be saved

## How to Run
```
pip install -r requirements.txt
python preprocessing.py
```

The script will download CIFAR-10, apply preprocessing, and save output images.
